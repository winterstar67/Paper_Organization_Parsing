- **날짜 로직 완전 재설계**: 공휴일/주말 복잡한 계산 제거, 데이터 기반 접근법 도입

**파일**: `3_parsing_meta_data.py`

#### 개발 배경:
- ✅ **Git Rollback**: 복합적 메타데이터 파싱에서 조직 정보 전용 추출로 전략 변경
- ✅ **새로운 접근법**: 규칙 기반 파싱의 한계를 극복하기 위한 AI 기반 솔루션 도입

#### 구현된 기능:
- ✅ **OpenAI GPT-3.5-turbo API 통합**: 자연어 처리를 통한 정확한 조직 추출
- ✅ **HTML 전처리 시스템**: 
  - Style 속성 제거 및 불필요한 태그 정리

- ✅ **구조화된 데이터 출력**: JSON 형식 조직 리스트 생성
